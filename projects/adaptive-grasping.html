02 — DEEP DIVE

Adaptive Grasping & Collision Recovery Strategy

C++ ROS2 MoveIt Motion Planning End-Effector Control

The Challenge

Standard manipulation pipelines fail rigidly when encountering unexpected obstacles near the target object in cluttered household environments. Traditional approaches hard-abort upon any contact or proximity violation, resulting in low success rates in real-world domestic settings where objects are densely packed and unpredictable.

The Implementation

Developed dynamic pick-maneuver logic that categorizes objects and adapts the end-effector approach in real time. Instead of hard-aborting upon minor environmental interference, the system intelligently recalculates palm angle openings and fallback trajectories, ensuring high success rates for grasping tasks in dense clutter.

Key Technical Details

  • Dynamic object categorization to determine optimal grasp strategy per target
  • Adaptive palm angle recalculation when collisions are detected — the gripper adjusts its approach vector rather than retreating
  • Fallback trajectory generation using MoveIt for alternative approach paths
  • Integration with the CPDS manipulation stack on ROS2 Humble
  • Trajectory caching system to reduce replanning latency during recovery maneuvers

Outcome

Significantly improved grasping success rates in cluttered household environments. The system handles edge cases — overlapping objects, soft fabrics, uneven surfaces — that previously caused hard failures, enabling Clutterbot's cleaning robot to operate reliably in real homes.
